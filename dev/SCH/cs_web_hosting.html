网页寄存

基本功能
网页流量 无限 无限
容量 1G 5G
FTP账户 2个 2个
网页预览 (全港首创)
分开预览和出街两个独立网页。设计师或程序编写员可以先将修改好的档案上载到预览网站给管理层过目，待审评后再上载到正式网站。 ✔ ✔
独立性
每个网页寄存都是使用独立区分的资源，即使其中一个网站有问题亦都不会影响另外一个。 ✔ ✔
数据备份
每日本地备份 ✔ ✔
每日异地备份 ✔ ✔
技术支援
asp.net 3.5, asp.net 2, asp.net 1.1, asp, Linq & Ajax, php4/php5, html ✔ ✔
数据库 access, mssql, mysql   ✔
无 WWW 网址支援
访客读取网址，同时支援在域名前面有和没有www ✔ ✔
一年送1次网站更新
简单文字以及图片更新，但不包括新增网页、图片的后期制作以及程序更改。每次更新页数最多5页，否则需收取额外费用。 ✔ ✔
访客统计
利用Google Analytics 的强大功能，除了可以记录访客人数区域分布和内容点击率，更加可以追踪到达网页的来源包括从各种搜索引擎、推荐网页连结、电邮及网上付款广告等。 ✔ ✔
