何谓 Microsoft Azure？
何谓 Azure？简单地说，就是 Microsoft 的云端平台：一个成长中的整合服务集合，包括运算、存储器、数据、网络及应用程序等，可协助您加快脚步、处理更多工作及节省成本。但这还只是轻描淡写。以下是 Azure 的其他描述…。
