Microsoft SharePoint 服务

甚么是Microsoft SharePoint Online?
SharePoint Online 提供了 SharePoint 的各项强大功能，方便用户随时随地利用各种装置上的 SharePoint行动应用程序共享、管理及搜寻信息与资源。 另外SharePoint Online支援无限量用户和免费提供公司域名，最紧要的是享有微软提供的99.9%上线时间保证，保证你不会因上线问题影响工作进度。
Microsoft SharePoint Online的优胜之处

共享： 企业员工可在 SharePoint 同时与其他人共享一份文件，文件内容会同步更新，方便员工之间分享想法和交流合作。
跨装置使用的能力： 您可以随时随地利用各种装置上的 SharePoint 行动应用程序存取您的 SharePoint 新闻摘要并与其互动。
没代码： SharePoint支援没代码构建应用程序，即使不是熟练的技术人员，也可以使用SharePoint构建适合自己企业的内部网络。
安全性： SharePoint 可以被视为一个安全位置来储存、组织和共享信息。只需要简单按几下，便可以把文件发布，与组织内外的人员共享。
节省开支： 在 SharePoint 的基础上建立高效能、具规模及管理功能的内部网站，能降低降底企业的基础结构成本，企业可以花更多时间进行创新活动，而非管理基础结构。
企业级的可靠性及标准： 将您的数据代管在分散于各地的数据中心，提供您持续性的数据备份、顶级灾难复原能力，以及全天候监控服务器的专家小组，来协助您保障数据的安全。
